## 🎯 What is this?

An **Excel Data Cleaner and Analyzer** that automatically detects phone numbers, purchase amounts, cleans data, calculates risk scores, segments customers, and generates business insights with color-coded reports.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔍 **Smart Detection** | Automatically finds phone and purchase columns |
| 🧹 **Data Cleaning** | Removes duplicates, fixes phone numbers |
| 📊 **Customer Segmentation** | Premium, VIP, Regular, Low categories |
| ⚠️ **Risk Scoring** | High/Medium/Low risk based on data quality |
| 📈 **AI Insights** | Business health score (0-100) with grade |
| 🎨 **Color-Coded Output** | Highlights rows by type and risk |
| 🔒 **License Protection** | Built-in license key system |
